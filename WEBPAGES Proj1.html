In the "Snake/ObamaIcon" project, Gabrielle worked to make an image with a similar coloring format to Obama's infamous blue and red "hope" campaign picture.

The instructions for the "Snake/ObamaIcon" project were...

-You can get a similar effect if you look at each pixes RGB, or red green blue, values. RGB values work like light. If a color has high values for all three colors, it is close to white. If there are low values, its close to black. If you add all the values together, you get a number that correlates to how light the color is. Lighter colors get colorized yellow. Medium/light colors get coded as light blue. Medium/dark colors are red and very dark colors get coded as dark blue.-
